# Caza — All In One Interactive Furniture Room Planner

Ayubowan ! Caza is a All-in One Room Planner Web application with 2D / 3D Room Experience. 


## Features

- User Login and Sign Up - Manage Designs and Client easily using cetralized dashbaord. 
- 3D Room Planner - Immersive 3D View of Room design.
- 2D Room Canvas - A shape-based 2D floor plan editor allows users to place, move, rotate, resize and delete furnitures.
- Furniture Panel - Dedicated Panel for Select Furnitures with Real time preview, Search Furnitures, Customize Room Settings.
- Properties Panel - Dedicated Panel for Edit Furniture Properties such as colours, scale and opacity.
- Cinematic Capture - Create Cinematic Video of Room Design.
- First Person Walk Mode - Allow users to walk through the room in 3D ( First Person View )
- Undo / Redo - Undo and Redo edits without fear of lossing current works.
- Furniture Placement Validation - User will get overlapping toast when adding more furniture in same place. 
- Responsive Design - Mobile and Desktop Responsive Web Application
- Save / Load Desins - User can save and load designs easily. 
- Room Settings - User can set own room width, depth, height, and Wall, floor , celing colours.




## Tech Stack

- Next Js - Framework
- Typescript - Language
- Tailwind - UI Styling
- Shadcn - UI Components
- Framer Motion - Animation
- Lucid React - Icon Set
- Three JS - WebGL 3D Engine
- React Three Fiber - React Rendere
- Drei - Helpers for React Three Fiber
- React Three Post Processing - Post proccesing for 3D Scene
- Konva - 2D Canvas Rendering
- Supabase - Database and Authentication

## 🚀 Getting Started

### Prerequisites

- **Node.js** v18 or later
- A [Supabase](https://supabase.com) project (for auth & database)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/hci.git
cd hci

# 2. Install dependencies
npm install

# 3. Configure environment variables
cp .env.example .env.local
# Fill in your Supabase URL and anon key (see Environment Variables below)

# 4. Start the development server
npm run dev
```
